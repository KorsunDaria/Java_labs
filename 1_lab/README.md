Инструкция к игре
1. Настройка (Setup)

При запуске программа предложит ввести параметры сложности:

Code length: Длина секретного числа (рекомендуется 4).

Max attempts: Сколько раз вы можете ошибиться.

Time limit: Время в секундах, отведенное на один ввод.

2. Игровой процесс

Вводите комбинацию цифр в консоль. После каждого хода программа выводит результат:

Bulls (Быки) — количество цифр, которые угаданы верно и стоят на своих местах.

Cows (Коровы) — количество цифр, которые есть в секретном коде, но стоят на неверных позициях.

Пример: > Секретный код: 1234. Ваш ввод: 1350.
Результат: Bulls=1 (цифра 1), Cows=1 (цифра 3).

3. Завершение

Вы выигрываете, если количество "Быков" равно длине кода.

Вы проигрываете, если закончатся попытки или если вы не успеете ввести число в установленное время (попытка сгорает).

Логирование
Все ваши игры записываются в файл game.log в корневой папке проекта. В логах фиксируется:

Дата и время начала игры.

Загаданное число.

Каждый ваш ход и его результат.

Итог партии (победа или поражение).

Для запуска программы: 
java -jar target/bulls-and-cows-1.0-SNAPSHOT.jar
